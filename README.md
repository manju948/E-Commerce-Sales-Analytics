# 📊 E-Commerce Sales Analytics

A complete SQL-based data analytics project that analyzes e-commerce sales data to generate meaningful business insights. This project demonstrates database design, data cleaning, SQL analysis, and an interactive frontend dashboard for visualizing key metrics.

---

## 🚀 Features

- SQL Server Database Design
- Data Cleaning & Validation
- Basic, Intermediate & Advanced SQL Queries
- Customer Analysis
- Product Performance Analysis
- Revenue Analysis
- Sales Trend Analysis
- Interactive Dashboard
- Responsive Website
- Professional UI with Animations
- Screenshot Gallery
- Business Insights

---

## 🛠️ Tech Stack

- SQL Server
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js
- AOS (Animate On Scroll)
- Bootstrap Icons
